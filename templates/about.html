About

Intelligent Deduper was created to make finding duplicate records easier. Instead of comparing records using a direct comparison of two fields, Intelligent Deduper makes use of machine learning to recognize similar records. Users upload their data then train Intelligent Deduper on what pairs of records are considered duplicates. This allows differences due to misspellings, different naming conventions (Rd. versus Road) and nicknames are caught and considered duplicates.

All of the machine learning logic is done using the Dedupe Python API.

This application was built for a final project in Harvard's introductory computer science course CS50x, offered free through Edx.org. Special thanks is given to all CS50 staff and professor David Malan.
